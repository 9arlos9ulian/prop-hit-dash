# prop-hit
Hit Dugout personal use
MLB batter analytics + bet tracker · not an autobetting system
what this is
A personal analytics dashboard for MLB hit prop bets. Each night it scores every starting batter with a probability of recording at least one hit, based on their recent form, the opposing pitcher, and game context. Output is a calibrated probability — not a pick, not a recommendation. You decide whether the number is worth the market price.
tabs
Tonight
Full batter pool sorted by hit probability. Shows recent streak, HOT/COLD badge, pitcher matchup, and BvP or career splits. Filter by slot, game, or streak. Build a parlay slip with the bet tracker.
Slate
Game-level view. Top hitters per team, pitcher matchup grades, game times, final scores.
Bets
Bet tracker with bankroll, P&L, and closing-line value per pick. Settled automatically each morning.
History
Logged picks vs outcomes. Daily scorecard, calibration buckets, team over/under performance.
measuring edge
Two checks must pass before treating the model as useful:
01
Calibration — when the model says 70%, those batters should actually get a hit ~70% of the time. Check via History → Calibration Buckets.
02
Closing-line value — beating the closing line consistently is the real signal. ROI on small samples is noise. Track it in the Bets tab over several weeks before drawing conclusions.
data via baseballsavant.mlb.com · statsapi.mlb.com · no bets placed automatically
